# Project w/ Cypress

## Project setup
```
npm install
```

### Database info
```
Complete the information related to your database (db name, password) in the database.js file that is in the ../server directory
```

### Run the Back-end
```
npm run server
```

### Run the Front-End with hot reload
```
npm run serve
```

### Open Cypress 
```
npx cypress open
```

### Compiles and minifies for production
```
npm run build
```
